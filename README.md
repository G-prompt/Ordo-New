# ORDO Prototype

This is a lightweight prototype of the ORDO product (Human Capital Operating System). It includes a static frontend and a small Express prototype API server with an in-built authentication flow for demo purposes. Built by Team Turing with polished page flows, improved contact validation, and a friendly 404 fallback.

## Quick start

Install dependencies and run:

```bash
npm install
npm run dev
```

Then open http://localhost:3000 in your browser.

For a production build:

```bash
npm run build
npm start
```

## What is included

- `server.js` — Express backend with auth, onboarding, career roadmap, projects, portfolio, verification, and opportunity endpoints.
- `index.html` — Root Vite entrypoint for the React/Tailwind frontend.
- AI proxy support via `VUYAR_API_KEY` (or `GEMINI_API_KEY` for compatibility) for `/api/v1/ai/assist`.
- `src/` — React application source for the ORDO interface.
- `tailwind.config.js` / `postcss.config.cjs` — Tailwind and PostCSS build configuration.
- `dist/` — Production frontend build output generated by `npm run build`.
- `API_DOC.md` — API documentation for the prototype.
- `src/pages/NotFound.jsx` — Friendly 404 fallback page for invalid routes.

## AI assistant setup
To enable the AI assistant, set `VUYAR_API_KEY` (or `GEMINI_API_KEY` for backward compatibility) in your environment and restart the server. This project is built by Team Turing and the assistant is powered by AI.

Example (Linux / macOS):
```
export VUYAR_API_KEY="sk-..."
npm run dev
```

Example (Windows PowerShell):
```
$env:VUYAR_API_KEY="sk-..."
npm run dev
```
